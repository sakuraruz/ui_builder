# Архитектура проекта

Каркас создан по техническому заданию из корневого `README.md`.

## Основные области

- `src/main` — главный процесс Electron, работа с окнами, файлами проекта и экспортом.
- `src/preload` — безопасный мост между Electron main и renderer.
- `src/renderer` — интерфейс визуального конструктора.
- `src/shared` — общие типы, константы и схемы данных.
- `tests` — будущие unit/e2e тесты.
