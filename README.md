## 1 Цель

Создание десктопного приложения для визуального проектирования веб-интерфейсов методом drag-and-drop с возможностью экспорта в чистый HTML/CSS код и настройкой всех визуальных свойств элементов.

## 2 Границы

### 2.1 Включено

1. Визуальный редактор с холстом для размещения элементов
2. Палитра базовых HTML-элементов (кнопки, текстовые блоки, изображения, карточки, формы)
3. Система drag-and-drop для добавления элементов на холст
4. Редактор свойств для настройки внешнего вида
5. Поддержка кастомизации CSS-свойств: цвета (текста, фона, границ), типографика (шрифт, размер, жирность, выравнивание), отступы (margin, padding), сглаживание углов (border-radius), тени (box-shadow, text-shadow), градиенты
6. Генерация чистого HTML/CSS кода
7. Сохранение проектов в собственном формате (.fld)
8. Экспорт в HTML файл
9. Масштабирование холста
10. Выравнивание элементов (по левому краю, по центру, сетка)

### 2.2 Исключено

1. Редактирование HTML кода вручную
2. Интерактивность/JavaScript на экспортируемой странице
3. Адаптивная верстка под мобильные устройства
4. Drag-and-drop между проектами
5. Импорт из Figma/Photoshop
6. Совместная работа нескольких пользователей

### 2.3 Ограничения на элементы

| Параметр | Минимум | Максимум | Обоснование |
|---|---|---|---|
| Количество элементов | 0 | 10³ | Нижняя граница — пустой холст; верхняя граница — производительность рендеринга |
| Размер холста (px) | 800 × 600 | 5000 × 5000 | Минимум — стандартное разрешение; максимум — ограничение памяти |
| Глубина вложенности | 1 | 10 | Избежание экспоненциального роста сложности парсинга |

## 3 Функционал

### 3.1 Основные компоненты интерфейса

#### Палитра элементов

1. Набор базовых блоков: кнопка, текст, изображение, карточка, контейнер, форма, поле ввода
2. Каждый элемент имеет иконку и название
3. Элементы перетаскиваются на холст

#### Холст

1. Область для размещения элементов
2. Масштабирование
3. Сетка для выравнивания (шаг 10px)
4. Выделение элементов кликом
5. Перемещение элементов мышью
6. Изменение размера через маркеры
7. Множественное выделение
8. Выравнивание выделенных элементов (по левому краю, по центру, по правому краю)

#### Инспектор свойств

Динамически обновляется в зависимости от выбранного элемента:

##### Вкладка "Стиль":

1. Цвет текста
2. Цвет фона
3. Шрифт
4. Размер шрифта
5. Жирность
6. Выравнивание текста
7. Сглаживание углов
8. Тень
9. Прозрачность

##### Вкладка "Геометрия":

1. Позиция X, Y (в px)
2. Ширина, Высота (в px)
3. Отступы margin/padding (по 4 стороны)

##### Вкладка "Дополнительно":

1. ID элемента
2. CSS классы
3. Alt текст (для изображений)
4. Ссылка (для кнопок)

### 3.2 Управление проектом

1. Создать проект - новый чистый холст
2. Открыть проект - загрузка .fld файла
3. Сохранить проект - сохранение в .fld
4. Сохранить как - выбор места сохранения
5. Экспорт в HTML - генерация HTML+CSS файла
6. Экспорт в HTML с просмотром - экспорт и открытие в браузере

## 4 Технологии

JavaScript (Node.js), Electron.js, HTML5, CSS3
